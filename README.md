## Customer Churn Prediction App: README
# Overview
The Customer Churn Prediction App predicts the likelihood of customer churn using inputs like geography, age, and account metrics. Built for businesses, it helps identify and retain at-risk customers.

# Features
Interactive Interface: Built with Streamlit for easy navigation.
Real-time Predictions: Instant churn probability based on user inputs.
Flexible Inputs: Covers key factors like credit score, balance, and activity status.
# Setup
Clone the repository and install dependencies:
git clone <repository-url>  
pip install -r requirements.txt  
Place model.h5, label_encoder_gender.pkl, onehot_encoder_geo.pkl, and scaler.pkl in the project folder.
Run the app:
streamlit run app.py  
# Usage
Enter customer details (e.g., geography, gender, credit score).
View churn probability and prediction result.
Use insights for customer retention strategies.
